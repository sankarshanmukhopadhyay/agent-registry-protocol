# Repository Positioning

The Agent Registry Protocol is the normative protocol layer for resolving agent identity, relationships, authority, assurance, status, evidence, and governance.

It is designed to remain transport-neutral and implementation-neutral.

Potential companion layers include:

- semantic and relational meta-models;
- machine-readable trust-artifact schema repositories;
- directory and resolution protocols;
- agent communication transports;
- policy decision and enforcement systems;
- domain-specific assurance profiles; and
- governance artifact and pressure-test corpora.

Companion implementations may adopt this protocol without transferring ownership of their transport, schema, governance, or execution models to this repository.
