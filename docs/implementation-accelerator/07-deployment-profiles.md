# Deployment profiles

| Profile | Purpose | Persistence | Authority boundary | Evidence expectation |
|---|---|---|---|---|
| Local | Developer learning | Optional | Single operator | Quickstart report |
| Single organisation | Internal evaluation | Required | One accountable entity | Lifecycle and decision receipts |
| Pilot | Controlled real-world trial | Required, backed up | Named governance authority | Readiness report and incident process |
| Federated pilot | Cross-registry evaluation | Required per operator | Explicit recognition agreements | Discovery, provenance and replay evidence |
| Production | Operational service | HA and recoverable | Enforced delegated authority | Continuous monitoring and audit retention |

Production selection is not achieved by changing a label. It requires satisfying the controls in [production hardening](09-production-hardening.md).
