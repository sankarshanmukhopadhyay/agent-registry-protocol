# End-to-End Scenarios

The scenarios below are executable governance narratives. Each identifies authority, enforcement, evidence and revocation outcomes.

| Scenario | Profile | Primary evidence |
|---|---|---|
| Informational agent | A | identity, description digest, lifecycle history |
| Enterprise workflow | B | typed operator, assurance and execution receipt |
| Bounded procurement | C | authority envelope, decision receipt, cascade revocation |
| Healthcare access | D | purpose/jurisdiction constraints and protected evidence |
| Public-service eligibility | D | policy version, appeal and restoration |
| Fiduciary analysis | D | explicit prohibitions and accountable entity |
| Operator change | B-D | relationship transition and assurance review |
| Compromised runtime | B-D | deployment quarantine and enforcement event |
| Stale projection | A-D | lag disclosure and indeterminate result |
| Federation conflict | D | recognition policy and conflict evidence |
| Emergency suspension | D | rapid status event, appeal and restoration |
| Delegated action | C-D | end-to-end resolution, decision, execution and audit |
