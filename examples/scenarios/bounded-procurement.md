# Bounded Procurement Agent

## Flow

Register identity → establish typed relationships → publish content-addressed description → resolve current status → evaluate applicable assurance or authority → produce evidence → apply lifecycle change → reconstruct point-in-time state.

## Governance assertions

Registration does not create authority. Every material claim identifies a competent issuer. Revocation or suspension identifies scope, effective time, enforcement targets and appeal route.

## Evidence

The implementation records source records, policy version, freshness, reason codes, decision receipt, event sequence and any execution receipt required by the selected profile.
